# fping-batch
fping batch process script with output file.

## What's this?
This is an batch linux script for fping, export csv table for analyze.

## How to use?
1. Install fping.
2. Download script.
3. Put your ip list file in the same folder and change it name as ips.txt.
4. Run script.
